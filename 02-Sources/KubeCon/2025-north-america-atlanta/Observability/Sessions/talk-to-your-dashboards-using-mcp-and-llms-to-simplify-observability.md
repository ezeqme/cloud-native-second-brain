## Descrição oficial

In large-scale application services, dashboards serve as windows into system health. But as the number of systems grow, so does the complexity. Dashboards can become outdated, and queries may drift over time. Large Language Models (LLMs) offer a way to simplify the management of dashboards and alerts using natural language prompts. However, integrating this capability into open-source systems is challenging to scale due to fragmented APIs. Enter, Model Context Protocol (MCP), which addresses this challenge by exposing observability primitives—dashboards, datasources, alerts—through a standardized schema that LLMs can understand. In this session, we'll demonstrate how MCP servers & clients can bridge LLMs with dashboard & alert management tools via natural language. You'll leave with practical techniques to integrate LLMs into your observability visualization stack like Perses—no custom wrappers, & no brittle integrations.
